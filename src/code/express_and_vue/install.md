## 安装nginx

```sh
apt update
apt install nginx
```

配置文件在`/etc/nginx/nginx.conf`

里面会引用其它的配置文件例如

```sh
        include /etc/nginx/conf.d/*.conf;
        include /etc/nginx/sites-enabled/*;
```

`sites-enabled`只是引用`sites-available`里的，里面有个`default`是用来测试nginx的，可以直接删掉，不然会占用80端口

设置开机启动

```sh
systemctl enable nginx
```
